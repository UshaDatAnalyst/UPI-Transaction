# UPI Transaction Analysis Dashboard

## Project Overview
This project analyzes UPI transaction data using Power BI to identify transaction trends, payment methods, user behavior, and financial insights.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV

## Key Insights
- Total Transactions
- Transaction Success Rate
- Top Payment Apps
- Monthly Transaction Trends
- State-wise Analysis
- Payment Category Analysis

## Dashboard Features
- KPI Cards
- Bar Charts
- Pie Charts
- Slicers & Filters
- Interactive Dashboard

- Power BI Dashboard (.pbix)
- Dataset
- Dashboard Screenshot
